# 有机合成标准操作规程：硝基还原 (Nitro Reduction)

**文档日期**：2026-1-21
**适用范围**：本规程适用于实验室将芳香族硝基化合物还原为苯胺衍生物。

---

## 1. 方法快速概览

| 方法 | 关键试剂 | 典型温度 | 适用场景 |
| :--- | :--- | :--- | :--- |
| **方法 A** | Pd/C + H₂ | **室温 (RT)** | 结构简单，无卤素，最干净。 |
| **方法 B** | Fe + NH₄Cl | **回流 (Reflux)** | 含卤素、醛酮，耐受性好。 |
| **方法 C** | SnCl₂ | **60℃ ~ 回流** | 特殊选择性，需防乳化。 |
| **方法 D** | Zn + AcOH | **0℃(起始) → 室温** | 反应快，位阻大，放热剧烈。 |

---

## 2. 详细操作步骤

### 方法 A：Pd/C 催化加氢 (标准法)
*   **反应温度**：**室温 (20~30℃)**
*   **注意**：严禁 Pd/C 干粉接触有机溶剂蒸汽，极易起火！

1.  **投料**：
    *   在反应瓶中加入底物。
    *   加入溶剂（==MeOH==, EtOH 或 EtOAc）。
    *   **安全操作**：先用氮气置换瓶内空气，或用少量水润湿 Pd/C，再将 Pd/C (5%-10% wt) 小心加入。
2.  **置换**：抽真空，通入氢气，重复置换操作 3 次。
3.  **反应**：在氢气球 (1 atm) 氛围下，**室温**剧烈搅拌 2-12 小时。
4.  **后处理**：
    *   通过硅藻土 (Celite) 抽滤除去钯碳。
    *   用溶剂洗涤滤饼。
    *   旋干滤液即得粗品。

### 方法 B：铁粉还原 (Fe/NH₄Cl - 经典耐卤素)
*   **反应温度**：**回流 (Reflux, 约 75~80℃)**
*   **试剂比例**：`底物 : Fe : NH₄Cl = 1 : 3~5 : 3~10 (摩尔比)`

1.  **投料**：
    *   烧瓶中加入底物、EtOH/H₂O (v/v = 4:1) 和 NH₄Cl。
    *   开启搅拌并加热至**回流状态**。
2.  **加铁粉**：
    *   在回流状态下，**分批**加入还原铁粉（防止剧烈放热冲料）。
3.  **反应**：保持**回流** 1-4 小时，TLC 监测。
4.  **后处理**：
    *   **趁热过滤**（硅藻土），用热乙醇大量洗涤（防止产物在冷时析出或被铁泥吸附）。
    *   旋去有机溶剂，水相调碱后萃取。
    *   可直接旋掉乙醇，过滤后使用EA萃取过柱获取教纯产物。

### 方法 C：二氯化锡还原 (SnCl₂)
*   **反应温度**：**60℃ 至 回流**
*   **试剂比例**：`底物 : SnCl₂·2H₂O = 1 : 3~5 (摩尔比)`

1.  **投料**：底物溶于 EtOH 或 EtOAc，加入 SnCl₂·2H₂O。
2.  **反应**：加热至 **60℃ 或回流**，通常需加少量浓 HCl 催化。
3.  **后处理**：
    *   推荐加入饱和**酒石酸钾钠**溶液破除乳化，搅拌至分层清晰后萃取。

### 方法 D：锌粉还原 (Zn Powder)
*   **反应温度**：**冰浴 (0℃) 起始，随后升至室温**
*   **试剂比例**：`底物 : Zn = 1 : 5~10 (摩尔比)`

**操作步骤 (以 Zn/AcOH 为例)**：
1.  **溶解与降温 (关键)**：
    *   将底物溶解在冰乙酸 (AcOH) 中。
    *   **务必冰浴冷却**：将反应瓶置于 **0-5℃** 冰水浴中。
2.  **投料**：
    *   在**低温**和剧烈搅拌下，**分批、缓慢**加入锌粉。
    *   *警告：反应放热非常剧烈，若不降温或加料过快，极易喷料！*
3.  **反应**：
    *   加完锌粉后，撤去冰浴，自然升至**室温 (RT)** 搅拌。
    *   通常 0.5-2 小时即可反应完全。
4.  **后处理**：
    *   过滤除去未反应的锌粉。
    *   旋去大部分乙酸。
    *   加入饱和 NaHCO₃ 调节 pH 至弱碱性，乙酸乙酯萃取。

## 3. 产物保存与注意事项

1.  **氧化敏感性**：芳香胺类化合物（尤其是富电子的）极易被空气氧化变红或变黑。
2.  **操作建议**：
    *   旋蒸温度不宜过高（建议 < 45℃）。
    *   得到产物后应尽快投入下一步反应。
    *   若需保存，请置于**低温、避光、充氮气**密封保存。
