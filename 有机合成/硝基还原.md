# 有机合成标准操作规程：硝基还原 (Nitro Reduction)

**文档日期**：2026-1-12
**适用范围**：本规程适用于实验室将芳香族硝基化合物还原为苯胺衍生物。

---

## 1. 方法选择指南

| 底物特征                | 推荐方法                | 关键优势             |
| :------------------ | :------------------ | :--------------- |
| **结构简单，无卤素**        | **方法 A (Pd/C 加氢)**  | 后处理最干净，产率高，操作简便。 |
| **含卤素 (Cl, Br, I)** | **方法 B (Fe/NH₄Cl)** | 不会发生脱卤副反应，工业首选。  |
| **含醛基/酮基/怕酸**       | **方法 B (Fe/NH₄Cl)** | 反应条件温和（近中性）。     |
| **对酸稳定/特殊选择性**      | **方法 C (SnCl₂)**    | 反应彻底，但后处理需防乳化。   |

---

## 2. 详细操作步骤

### 方法 A：Pd/C 催化加氢 (标准法)
**注意**：严禁Pd/C干粉接触有机溶剂蒸汽，极易起火！

1.  **投料**：
    *   在反应瓶中加入底物。
    *   加入溶剂（MeOH, EtOH 或 EtOAc）。
    *   **安全操作**：先用氮气置换瓶内空气，或用少量水润湿Pd/C，再将 Pd/C (5%-10% wt) 小心加入。
2.  **置换**：抽真空，通入氢气，重复置换操作3次。
3.  **反应**：在氢气球 (1 atm) 氛围下，室温剧烈搅拌 2-12 小时。
4.  **后处理**：
    *   通过硅藻土 (Celite) 抽滤除去钯碳。
    *   用溶剂洗涤滤饼。
    *   旋干滤液即得粗品（通常纯度极高）。

### 方法 B：铁粉还原 (耐卤素法)
**试剂比例**：底物 : Fe : NH₄Cl = 1 : 3~5 : 3~5 (摩尔比)

1.  **投料**：
    *   烧瓶中加入底物、EtOH/H₂O (v/v = 3:1) 和 NH₄Cl。
    *   开启搅拌并加热至回流。
2.  **加铁粉**：
    *   在回流状态下，**分批**加入还原铁粉（防止剧烈放热冲料）。
3.  **反应**：保持回流 1-4 小时，TLC监测。
4.  **后处理**：
    *   **趁热过滤**（通过硅藻土），用热乙醇或乙酸乙酯大量洗涤滤饼。
    *   旋去大部分有机溶剂。
    *   剩余水相用饱和 NaHCO₃ 调至弱碱性，用 EtOAc 萃取。
    *   干燥、旋干。

### 方法 C：二氯化锡还原 (SnCl₂)
**试剂比例**：底物 : SnCl₂·2H₂O = 1 : 3~5 (摩尔比)

1.  **投料**：底物溶于 EtOH 或 EtOAc，加入 SnCl₂·2H₂O。
2.  **反应**：回流或加热至 60℃，通常需加少量浓 HCl 催化。
3.  **后处理 (破除乳化关键)**：
    *   **方法一 (推荐)**：加入饱和**酒石酸钾钠**溶液，剧烈搅拌 30-60 分钟至分层清晰，然后萃取。
    *   **方法二**：小心滴加 NaOH 溶液至 pH > 10，使锡盐溶解或沉淀颗粒化，再进行萃取。

---

## 3. 产物保存与注意事项

1.  **氧化敏感性**：芳香胺类化合物（尤其是富电子的）极易被空气氧化变红或变黑。
2.  **操作建议**：
    *   旋蒸温度不宜过高。
    *   得到产物后应尽快投入下一步反应。
    *   若需保存，请置于**低温、避光、充氮气**密封保存。
