# 上单BOC与双BOC保护方法总结

## 1. 上单BOC保护（单BOC化）

### 方法A：使用BOC酸酐（(Boc)₂O）
- **投料当量**：底物 : (Boc)₂O : 碱 = 1 : 1.1-1.5 : 2-3（当量）
- **常用碱**：三乙胺（Et₃N）
- **反应温度**：0°C至室温（20-25°C）
- **反应时间**：2-16小时
- **注意事项**：
  - 无水条件下反应效果更佳
  - 对空气敏感时需氮气/氩气保护
  - 若底物含多个官能团，注意BOC选择性
- **后处理**：
  - 反应完成后，加水淬灭，乙酸乙酯（EA）萃取
  - 有机相依次用稀HCl、饱和NaHCO₃、盐水洗涤
  - 无水Na₂SO₄干燥，减压浓缩得粗产物
  - 必要时通过柱层析（硅胶，EA/石油醚）纯化

### 方法B：使用BOC-ON（2-(叔丁氧羰基氧亚氨基)-2-苯乙腈）
- **投料当量**：底物 : BOC-ON : 碱 = 1 : 1.2-1.5 : 2-3
- **常用碱**：DBU、三乙胺
- **反应温度**：0°C至室温
- **注意事项**：适用于对碱敏感的底物，副产物为苯乙腈，易于去除
- **后处理**：与方法A类似，注意副产物极性较低，易通过柱层析分离

---

## 2. 上双BOC保护（双BOC化）

### 方法A：两步法
1. **第一步**：上第一个BOC（参考单BOC方法）
2. **第二步**：上第二个BOC（通常需调整条件）
   - **投料当量**：单BOC中间体 : (Boc)₂O : 碱 = 1 : 1.2-2.0 : 2-4
   - **反应温度**：室温至40°C
   - **注意事项**：
     - 第一步后需纯化中间体
     - 第二步可能需要更强碱（如NaH、n-BuLi）促进第二个BOC接入
     - 监测反应进程（TLC/HPLC），避免过度BOC化或副反应

### 方法B：一锅法双BOC化
- **投料当量**：底物 : (Boc)₂O : 碱 = 1 : 2.5-4.0 : 5-8
- **常用碱**：DMAP（催化量1.2eq）+ 三乙胺(3eq)或DBU or 只添加 DMAP:1.5eq 
- **反应温度**：室温50°C
- **反应时间**：6-24小时
- **注意事项**：
  - 适用于活性较高或需完全BOC保护的底物（如多胺类）
  - 严格控制温度，避免高温导致BOC基团脱落或分解
- **后处理**：
  - 反应液减压浓缩去除过量(Boc)₂O和溶剂
  - 剩余物溶于EA，依次用稀酸、碱、盐水洗涤
  - 干燥、浓缩后柱层析纯化（梯度洗脱）

---

## 3. 通用注意事项

### 反应条件优化
- **温度**：一般从低温开始，逐渐升温至室温；若反应慢，可升至40-50°C，但需监控副反应
- **溶剂**：常用二氯甲烷（DCM）、四氢呋喃（THF）、DMF等，需无水处理
- **监测**：建议使用TLC或HPLC跟踪反应，确保完全转化

### 后处理要点
1. **萃取**：使用合适极性溶剂（EA、DCM），注意乳化
2. **洗涤**：酸性洗涤去除碱性杂质，碱性洗涤去除酸性杂质
3. **干燥**：无水Na₂SO₄或MgSO₄干燥后过滤
4. **纯化**：
   - 粗产物可通过柱层析（硅胶，EA/PE或DCM/MeOH）
   - 若产物为固体，可尝试重结晶

### 储存与稳定性
- BOC保护产物通常对空气稳定，但建议低温（-20°C）干燥保存
- 避免强酸、强碱条件，以防BOC基团脱落
